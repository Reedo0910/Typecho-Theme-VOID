# Typecho Theme VOID (Special Edition) ~~(Hack)~~

> Typecho VOID主题的个人修改版

## 适用版本

VOID 3.4.0

## 拓展功能

* WebP图片支持

  * **使用方法：**
  
    * 在Typecho的编辑器中插入支持所有浏览器的图片（如jpg、png等）

    * 准备一张和插入图片同名、同路径（即除了后缀名以外都相同）的WebP格式的图片。（该图片不需放到编辑器中）

    * 文章发布后，主题即会对支持WebP的浏览器启用这张WebP图片。若相应的WebP图片不存在，或浏览器不兼容WebP（如Safari、IE等），主题则会启用文章中插入的图片。

  * **适用人群：**

    * 喜欢追求新鲜玩意儿的人士

    * 希望减少流量开支的人士
